# Tech_Storage

기술 자료 수집 — 관심 있는 기술 주제를 마크다운으로 정리해 축적하고, 슬랙 등에 공유하기 위한 저장소입니다.

## 구조

```
기술동향/
  YYYY-MM-DD-주제.md   # 날짜순으로 쌓이는 개별 기술동향 문서
```

## 작성 규칙

- 파일명: `YYYY-MM-DD-주제-슬러그.md`
- 각 문서는 한 줄 요약 → 핵심 내용 → 참고자료(공식 문서 우선) 순으로 구성
- 슬랙에 그대로 복사-붙여넣기 가능하도록 표준 마크다운(헤더, 리스트, 링크)만 사용
